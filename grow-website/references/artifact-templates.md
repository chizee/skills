# Artifact Templates — grow-website

Skeletons this metaskill creates in the user's docs/ folder. Extend-only artifacts are governed by the section headings named in each phase; the three below (MARKETING.md, WEBSITE.md, EXPERIMENTS.md) are the ones grow-website most often authors from scratch, so their full skeletons are copied here. Read before writing, create only if missing, and preserve any sections another journey already wrote.
